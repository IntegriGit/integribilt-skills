# autonomous-agent-patterns

## What I do
I provide design patterns for building autonomous coding agents inspired by Cline and OpenAI Codex. I offer patterns for core agent architecture, tool design, permissions, safety, browser automation, and context management (MCP integration, checkpoints).

## How to use me
Use me when you are building autonomous AI agents, designing tool/function calling APIs, implementing permission and approval systems, creating browser automation, or designing human-in-the-loop workflows. Follow my provided architectural blueprints and safety best practices.

## Tools I use
- Agent Loops and Checkpoint Managers
- MCP (Model Context Protocol) Server Integration
- Browser Automation tools (Playwright/Puppeteer)
- LLMs (Multi-Model Architectures)
