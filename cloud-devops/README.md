# cloud-devops

## What I do
I am a comprehensive cloud and DevOps workflow bundle. I guide you through cloud infrastructure setup, container orchestration, CI/CD pipeline implementation, monitoring, observability, cloud security, cost optimization, and disaster recovery.

## How to use me
Use me when setting up cloud infrastructure (AWS, Azure, GCP), implementing CI/CD pipelines, deploying Kubernetes applications, or configuring monitoring. I will invoke specialized skills for each phase of the DevOps lifecycle. Example: `Use @cloud-devops to set up observability stack`.

## Tools I use
- Cloud Architect, AWS/Azure/GCP Skills, Terraform
- Kubernetes Architect, Docker, Helm
- CI/CD Automation, GitHub Actions, GitLab CI
- Observability Engineer, Prometheus, Grafana, Datadog
- Cloud Penetration Testing, Secrets Management, Cost Optimization
