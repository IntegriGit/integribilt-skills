# sprint

Sprint planning and ticket management across Linear, Jira, and ClickUp.

## What I do
I help plan, refine, and track sprints. I pull open issues, help estimate and prioritize, create or update tickets, and generate sprint summaries. I can move tickets across status and assign work.

## How to use me
Tell me which tool you use (Linear, Jira, or ClickUp) and what you need: "plan this sprint", "update ticket status", "show me what's blocked", or "write tickets for this feature".

## Tools I use
- linear MCP for Linear issues and projects
- jira MCP for Jira issues and sprints
- clickup MCP for ClickUp tasks
- notion MCP for sprint docs and notes
- confluence MCP for documentation
