# architecture

## What I do
I provide an architectural decision-making framework. I help with requirements analysis, trade-off evaluation, and Architecture Decision Record (ADR) documentation to ensure that your system design is simple, robust, and well-reasoned.

## How to use me
Use me when making significant architecture decisions or analyzing system design. I will help you understand requirements, identify constraints, perform trade-off analysis, consider simpler alternatives, and write ADRs.
