# antigravity-skill-orchestrator

## What I do
I am an intelligent meta-skill that enhances an AI agent's ability to handle complex, multi-domain tasks. I evaluate task complexity, dynamically select the best combination of skills for a given problem, and track successful combinations over time using the `agent-memory-mcp` skill to build institutional knowledge.

## How to use me
Invoke me within the Antigravity IDE when dealing with a complex objective that requires combining several skills. I will break down the task, search memory for similar challenges, assemble the right team of skills, and execute the task efficiently. Make sure the `agent-memory-mcp` skill is installed and running. Example invocation: `@antigravity-skill-orchestrator Please build a comprehensive dashboard integrating fetching live data, an interactive UI, and performance optimizations.`

## Tools I use
- agent-memory-mcp
- Relevant specialized skills (e.g., @react-patterns, @nodejs-backend-patterns)
