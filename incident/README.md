# incident

Structured incident response for the IntegriBilt infrastructure.

## What I do
I lead you through incident triage using a structured P1/P2/P3 framework. I check Zabbix alerts, query logs in Loki/Elasticsearch, review Grafana dashboards, and help coordinate resolution. I document findings and generate post-incident summaries.

## How to use me
Describe the symptom or paste the alert. I will immediately triage severity, identify blast radius, and begin systematic diagnosis.

## Tools I use
- zabbix MCP for alert and host status
- grafana / prometheus / loki MCP for metrics and logs
- elastic MCP for log search
- pagerduty / opsgenie MCP for escalation
- slack MCP for incident channel communication
