# antigravity-workflows

## What I do
I orchestrate multiple Antigravity skills through guided workflows for tasks such as shipping a SaaS MVP, running web security audits, building AI agent systems, and implementing browser automation/E2E QA.

## How to use me
Use me when you want to combine several skills without manually selecting each one, or when your goal is multi-phase (plan, build, test, ship). I will propose matching workflows, ask you to choose one, and execute it step-by-step, verifying completion criteria before moving to the next step. Invoke me with prompts like: `Use @antigravity-workflows to run the "Ship a SaaS MVP" workflow for my project idea.`

## Tools I use
- docs/WORKFLOWS.md
- data/workflows.json
- Relevant specialized skills (e.g., concise-planning, brainstorming, workflow-automation, verification-before-completion)
